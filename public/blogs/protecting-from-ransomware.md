# Protecting Your Computer from Ransomware

Ransomware attacks are on the rise, affecting individuals and businesses alike. This guide will help you protect your data and know what to do if you become a victim.

## What is Ransomware?

Ransomware is malicious software that encrypts your files and demands payment (usually in cryptocurrency) to restore access. It can spread through:

- **Phishing emails** with malicious attachments
- **Compromised websites**
- **Infected software downloads**
- **Network vulnerabilities**

### 2. Backup Your Data

The **3-2-1 Backup Rule**:

| Copies | Storage Types           | Location           |
| ------ | ----------------------- | ------------------ |
| 3      | Total copies of data    | -                  |
| 2      | Different storage media | Local + External   |
| 1      | Offsite/Cloud           | Different location |

**Backup tips:**

- Automate your backups
- Test restore procedures regularly
- Keep offline backups (air-gapped)

### 3. Use Strong Security Software

Essential security layers:

1. **Antivirus/Anti-malware**

   - Windows Defender (built-in, good enough for most)
   - Malwarebytes (excellent second opinion scanner)

2. **Firewall**

   - Keep Windows Firewall enabled
   - Consider hardware firewall for businesses

3. **Email filtering**
   - Block executable attachments
   - Scan incoming files

### 5. Limit User Privileges

- Don't use admin accounts for daily tasks

### 6. Network Security

**For home users:**

- Use strong WiFi passwords (WPA3 if available)
- Change default router credentials
- Enable network encryption

**For businesses:**

- Segment your network
- Use VPN for remote access
- Implement Zero Trust architecture

## What To Do If Infected

### Immediate Steps

1. **Disconnect immediately**

   - Unplug network cable
   - Turn off WiFi
   - Isolate from other devices

2. **Don't pay the ransom**

   - No guarantee of recovery
   - Funds criminal operations
   - May be targeted again

3. **Document everything**
   - Screenshot ransom notes
   - Note the ransomware name
   - Record file extensions

### Recovery Options

1. **Check for decryption tools**

   - [No More Ransom Project](https://www.nomoreransom.org/)
   - May have free decryptors

2. **Restore from backup**

   - Clean system first
   - Restore from known-good backup

3. **Professional help**
   - Contact cybersecurity experts
   - Report to authorities

## Reporting Ransomware

Report incidents to:

- **Local police**
- **FBI IC3** (ic3.gov) in the US
- **CERT-In** in India
- Your country's cybercrime unit
