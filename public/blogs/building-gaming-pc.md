## Assembly Guide

### Step-by-Step

1. **Prepare workspace**

   - Clear, well-lit area
   - Anti-static precautions

2. **Install CPU**

   - Open socket lever
   - Align triangle markers
   - Don't force it!

3. **Install RAM**

   - Check manual for slot positions
   - Push until clips click

4. **Install M.2 SSD**

   - Insert at angle
   - Secure with screw

5. **Install CPU cooler**

   - Apply thermal paste (if needed)
   - Mount cooler securely

6. **Install motherboard in case**

   - Install I/O shield first
   - Use all standoffs

7. **Install GPU**

   - Remove slot covers
   - Insert and secure

8. **Connect power cables**

   - 24-pin motherboard
   - 8-pin CPU
   - GPU power

9. **Connect front panel**

   - Check motherboard manual
   - USB, audio, power switch

10. **Cable management**
    - Route cables behind motherboard tray
    - Use zip ties
